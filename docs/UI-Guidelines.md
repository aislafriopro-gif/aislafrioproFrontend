> **Proyecto:** Aislafriopro  
> **Versión:** V1.0  
> **Última actualización:** 18 de Julio 2026

---

# Objetivo

Este documento define la dirección visual oficial del proyecto **Aislafriopro**.

La interfaz del sistema estará basada en un análisis de diferentes referencias del sector industrial, seleccionando las mejores prácticas de navegación, experiencia de usuario, organización del contenido y presentación visual.

Este documento servirá como guía para el desarrollo de todas las vistas del proyecto.

---

# Filosofía Visual

La aplicación deberá transmitir:

- Profesionalismo
- Confianza
- Innovación
- Tecnología
- Orden
- Limpieza visual
- Facilidad de navegación

El objetivo es que cualquier usuario pueda identificar rápidamente los productos, comprender las soluciones ofrecidas por la empresa y solicitar una cotización de manera sencilla.

# Principios de Diseño

Toda la interfaz deberá cumplir con los siguientes principios:

- Navegación simple e intuitiva.
- Diseño limpio y moderno.
- Uso de imágenes de alta calidad.
- Espacios en blanco para mejorar la lectura.
- Información organizada por bloques.
- Componentes reutilizables.
- Llamados a la acción visibles.
- Enfoque Mobile First.
- Accesibilidad.
- Consistencia visual.

---

# Buenas Prácticas

- No saturar las pantallas con información.
- Priorizar la facilidad de uso.
- Mantener una misma línea visual en todo el proyecto.
- Mostrar evidencia del trabajo realizado mediante proyectos y casos de éxito.
- Facilitar siempre el acceso a la cotización y a los canales de contacto.
- Todas las nuevas interfaces deberán respetar esta guía visual.
